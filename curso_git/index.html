Git init ou Git clone?

git init - começar um repositório na pasta local (ou) git clone 'url.git' (endereço de onde se encotra o repositório na rede) ---------------------------------- Após realizar um 'git init' você poderá trabalhar no seu projeto normalmente. Poderá digitar 'git branch' para verificar a branch que está, ou até mesmo digitar 'git checkout' nome-da-branch para entrar na branch desejada. Após breves conclusões é recomendado realizar o 'git pull' para certificar-se de que está com o projeto atualizado e depois: git status > git add > git commit -m "breve_mensagem" > git push origin banch_name Concetar uma branch remotamente (exemplo): "git branch --set-upstream-to=origin/ curso_git"
